# Dutch Auction

This project implements the Dutch Auction mechanism to enable sales of tokens/ items. 

## Dependencies

* Truffle
* testrpc

## Installation

* `git clone https://gitlab.com/sidutta/dutch-auction-ethereum.git`
* `cd dutch-auction-ethereum`

## Run Tests

* `truffle test`

## Relevant files
* contracts/DutchAuction.sol
* test/dutchaution.js
